### Title
```
表1: Person

+-------------+---------+
| 列名         | 类型     |
+-------------+---------+
| PersonId    | int     |
| FirstName   | varchar |
| LastName    | varchar |
+-------------+---------+
PersonId 是上表主键
表2: Address

+-------------+---------+
| 列名         | 类型    |
+-------------+---------+
| AddressId   | int     |
| PersonId    | int     |
| City        | varchar |
| State       | varchar |
+-------------+---------+
AddressId 是上表主键
```

### Coding
```
select p.FirstName,p.LastName,a.City,a.State from Person p left join  Address a on p.PersonId=a.PersonId
```

因为不管是否匹配到空都要显示出来，所以用外联接。
